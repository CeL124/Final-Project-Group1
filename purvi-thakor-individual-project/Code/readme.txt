Please make sure to place the two code files in your working directory.
Run the codes in the following order in your working directory.

1) 02_pytorch_pre_processing.py

After running the 02_pytorch_pre_processing.py file a folder called svhn_pytorch will be created which will contain images in their respective label folder. 

2) 03_pytorch_model.py
This is the code that gives the best model.

Requirements file for the Pytorch environment is provided.
